# Security Policy

## Supported Versions

Security fixes are provided for the latest released version of each package:

- `allure_dart_commons`
- `allure_dart_test`
- `allure_flutter_test`

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Use GitHub Security Advisories for this repository, or contact Qameta through
the official channels listed at https://qameta.io.

Include as much detail as possible:

- Affected package and version.
- Steps to reproduce.
- Expected and actual behavior.
- Potential impact.
- Any known workaround.

We will review reports as quickly as possible and coordinate disclosure when a
fix is available.
