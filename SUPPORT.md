# Support

For questions, usage help, and troubleshooting, use the Allure Framework
community channels:

- Documentation: https://allurereport.org/docs/
- Questions and support: https://github.com/orgs/allure-framework/discussions/categories/questions-support
- General discussion: https://github.com/orgs/allure-framework/discussions/categories/general-discussion
- Announcements: https://github.com/orgs/allure-framework/discussions/categories/announcements

Use GitHub issues for confirmed bugs or actionable feature requests in this
repository.

Please do not use public issues to report security vulnerabilities. See
`SECURITY.md` instead.
